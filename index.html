Getting Started with GraphQL and Apollo.

Free tutorial for developing a GraphQL web application.

by: Therin Mody

Introduction

Welcome to my tutorial on developing a full stack GraphQL application with Apollo technologies.

Our application will contain one simple feature, fetching data from the sports data io API and display it on our screen.

Here is the final outcome we aim to achieve:



When we build this feature, we will take a “schema-first” design approach. There are three major steps in using the schema-first design approach:

1. Define the Schema: We identify the data our application will need, and position our schema to retrieve the data.

2. Server: The server will retrieve the data from our API and pass it along to the Client side of our application.

3. Client: The client will call on the server to retrieve the data and then render it to our screen.


This application will be written with Node.js on the backend and React on the frontend. If you are unfamiliar with these, don’t be alarmed. I have provided a Git repository for you to have an initial starting point and I will detail each step of the development process. This tutorial is designed to be a good start at becoming familiar with React and Node.js. However, it will mainly focus on the glory of using GraphQL with Apollo.
